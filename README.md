# cse-project
BMI Health Checker:

[A] Project Overview:

This is a simple command line project made in Python. The program calculates BMI (Body Mass Index) using the user's weight and height. It then shows whether the person is Underweight, Normal, Overweight, or Obese.
This project is created for beginner level and uses basic Python concepts only.

[B].Problem Statement:

Many people do not know how to calculate their BMI. Doing it manually is confusing. This program provides an easy way to calculate BMI and understand basic health status.

[C] Objectives:

Take weight and height from the user
Calculate BMI value
Display BMI result
Show health category
Allow the user to calculate BMI multiple times

[D] How the Program Works:

1 The user enters weight in kilograms.
2 The user enters height in meters.
3 The program calculates BMI using a formula.
4 It shows the BMI value and health status.
5 The user can repeat the process or exit.

[E] Technologies Used:

Python 3
Command Line Interface (CLI)

[F] How to Run the Program:

1 Make sure Python 3 is installed on your system.
2 Open terminal or command prompt.
3 Go to the project folder.
4 Run the following command:
python bmi_health_checker.py

[G] Features:

Simple and beginner-friendly
Fast BMI calculation
Displays clear health result
Handles incorrect input
