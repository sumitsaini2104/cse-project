# Project Statement

## Project Title
BMI Health Checker (Python Project)

## Project Description

The BMI Health Checker is a simple Python-based command line application that calculates Body Mass Index (BMI) using the userâ€™s weight and height. It helps users understand whether their weight is healthy or not by displaying categories like Underweight, Normal, Overweight, and Obese.

Many people find it difficult to calculate BMI manually, so this project provides an easy and fast solution using basic programming concepts.

## Purpose of the Project

The main purpose of this project is to:
- Make BMI calculation easy for everyone
- Help beginners understand Python basics
- Provide a simple health-checking tool

## Problem Statement

Manual BMI calculation is confusing and time-consuming for many people. There is a need for a simple system that can calculate BMI accurately and display the result in a user-friendly way. This project solves that problem by automating the process.

## Scope of the Project

This project focuses on:
- Taking user input for height and weight
- Calculating BMI using standard formula
- Displaying health category
- Allowing multiple calculations in one run

## Conclusion

The BMI Health Checker is a useful beginner-level Python project that combines learning with real-life application. It shows how simple code can be used to solve everyday health-related problems efficiently.
